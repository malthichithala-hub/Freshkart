FreshKart – Direct Farmer to Customer Marketplace
Project Description

FreshKart is a web-based marketplace that directly connects farmers with customers. The platform allows farmers to upload and manage their products while customers can browse products, add them to a cart, and place orders. The system removes middlemen and ensures fresh agricultural products reach customers directly from farmers.

Installation Steps

Clone the repository

git clone https://github.com/yourusername/freshkart.git

Navigate to the project folder

cd freshkart

Install required packages

pip install -r requirements.txt

Run the application

python app.py

Open browser and go to

http://127.0.0.1:5000
